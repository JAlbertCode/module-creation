Usage Instructions

  1. Select Model

    Choose a model from Hugging Face or use one of our examples

  2. Generate Module

    Click Generate to create your Lilypad module

  3. Deploy Module
    # Extract the files unzip lilypad-module.zip # Build and deploy docker build -t username/model-name . docker push username/model-name:latest # Run on Lilypad lilypad run username/model-name \ -i input_path=/workspace/input/file.ext \ -i input_text="Your text here"
