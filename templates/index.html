Usage Instructions

  1. Get Model URL

    Copy the URL from Hugging Face for any supported model type

  2. Generate Files

    Paste the URL and click "Generate Module Files"

  3. Deploy to Lilypad
    # Extract the files unzip lilypad-module.zip # Build and push Docker image docker build -t username/model-name . docker push username/model-name:latest # Test locally docker run -v $(pwd)/input:/workspace/input \ -v $(pwd)/output:/outputs \ username/model-name # Run on Lilypad lilypad run username/model-name \ -i input_path=/workspace/input/file.ext \ -i input_text="Your text here"
