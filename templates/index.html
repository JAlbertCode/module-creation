Quick Start Guide

  1. Paste Model URL

    Copy the URL from Hugging Face and paste it above

  2. Generate and Download

    Click Generate to download the module package

  3. Setup Local Environment
    unzip lilypad-module.zip cd lilypad-module
  4. Build and Run
    # Build Docker image docker build -t my-model . # Run locally docker run -v $(pwd)/input:/workspace/input \ -e INPUT_PATH=/workspace/input/input.txt \ my-model
  5. Deploy to Lilypad
    lilypad module deploy .
