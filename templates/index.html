Quick Start Guide

  1. Paste Model URL

    Copy the URL from Hugging Face and paste it above

  2. Preview Model (Optional)

    Click Preview to see how the model works with sample input

  3. Generate and Download

    Click Generate to download the module package

  4. Setup Local Environment
    unzip lilypad-module.zip cd lilypad-module
  5. Build and Run
    # Build Docker image docker build -t my-model . # Prepare input mkdir -p input # Add your input file to the input directory # Run locally docker run -v $(pwd)/input:/workspace/input \ -e INPUT_PATH=/workspace/input/input.txt \ my-model
  6. Deploy to Lilypad
    lilypad module deploy .
